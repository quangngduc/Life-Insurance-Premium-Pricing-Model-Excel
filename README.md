# Life-Insurance-Premium-Pricing-Model-Excel

## Objective

Develop a life insurance pricing model for an individual aged 27 seeking coverage until age 65.

## Features

- Salary projection
- Mortality modeling
- Premium calculation
- Present value discounting
- Sensitivity analysis
- Dashboard visualization

## Tools

- Microsoft Excel
- XLOOKUP
- Data Tables
- Financial Modeling

## Key Results

Calculated fair premium requirements under:
- Flat benefit design
- Salary-linked benefit design
